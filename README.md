Este proyecto es para hacerle un regalo a mi novia por nuestro aniversario, será uno de los regalos.
Ella sabe que he estado aprendiendo CSS con html y Javascript para mejorar mi lógica de programación y aprender nuevos lenguajes, vio como fue mi primer página web con simple texto y links que llevaban a otras páginas y en esa misma página le dije que ya no me acordaba ni de poner imágenes y que con un curso que estoy llevando aprendería eso de nuevo y muchas más cosas.
Para nuestro aniversario quería hacer alguna galería de fotos para recordar todo lo que hemos vivido este año juntos, pero si lo hacía en físico no me iba a caber ni el texto ni las fotos en alguna parte, asi que decidí hacer una página web para no tener ningun límite y que esté en internet para siempre y asi pueda ella visitar esa página cada vez que quiera.
Como es mi primer proyecto que haya requerido bastante tiempo (porque tuve que aprender css desde 0) los archivos no tienen tanto orden como quizás se podría llegar a tener en un proyecto profesional, el objetivo será seguir mejorando.
